# Exploratory Data Analysis (EDA) Portfolio – Python

## Overview

This repository contains multiple Exploratory Data Analysis (EDA) projects performed using Python. Each project focuses on exploring real-world datasets to uncover patterns, identify trends, detect anomalies, and generate meaningful business insights through statistical analysis and data visualization.

The projects demonstrate skills in:

* Data inspection and exploration
* Summary statistical analysis
* Missing value analysis
* Distribution analysis
* Outlier detection
* Categorical data analysis
* Correlation analysis
* Relationship analysis
* Time trend analysis
* Data visualization
* Business insight generation

## 📂 Projects Index

### 🎬 Movie Ratings Exploratory Data Analysis

**Status:** Completed

An end-to-end Exploratory Data Analysis was performed on a movie ratings dataset containing **449 movies** and **19 features**. The analysis explores ratings, popularity, genres, languages, review counts, and release trends using statistical techniques and visualizations.

**Key Analysis Performed:**

* Data Inspection
* Summary Statistics
* Missing Value Analysis
* Distribution Analysis
* Outlier Analysis
* Genre & Language Analysis
* Correlation Heatmap
* Relationship Analysis using Scatterplots
* Time Trend Analysis
* Top Movies Analysis
* Key Insights & Conclusion

**Tools Used:** Pandas, NumPy, Matplotlib, Seaborn
